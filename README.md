# Flip-dot display 3x5

## Features
- Modular flip-dot display architecture with daisy-chain support.
- UART host interface.
- RS485 bus.
- AT commands.
- Automatic node addressing.
- Bidirectional communication.
- Supports Arduino, ESP32, Raspberry Pi, and more.
